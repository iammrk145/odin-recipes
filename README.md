# odin-recipes

A simple HTML project built as part of The Odin Project curriculum.  
It contains a homepage and multiple recipe pages.

## Features

- Homepage linking to individual recipes
- Recipe pages include:
  - Dish image
  - Description
  - Ingredients list
  - Cooking steps

## Project Structure

odin-recipes/
├── index.html
├── recipes/
│ └── recipe-name.html
├── images/
│ └── image-name.jpg
└── README.md
